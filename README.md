Repo to store all stuff related with tests in Jenkins

Dependencies:

- git
- cpu-checker to check if kvm is supported
- qemu-kvm
- libvirt-bin
- bridge-utils
- virt-manager
- mkisofs
- vconfig
- java (JRE)
- EC2 Command Line Interface
- AWS CLI
- jq
- ruby
- jmeter

To install KVM you can follow this instructions:
https://help.ubuntu.com/community/KVM/Installation
